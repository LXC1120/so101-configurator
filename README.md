# 舵机配置工具（Web Serial）

这是一个**纯静态网页**，使用 **Web Serial API** 直接在浏览器里通过串口配置舵机（无需安装 Python/软件）。

## 功能

- **流水线自动配置 ID**：设置起始/结束 ID 后点击“开始流水线”，之后只需要**不断插拔舵机**，网页会自动检测“插入”并依次写入 ID。
- **单独改号**：保持你原 `py` 脚本的流程（解锁→关扭力→写 ID→上锁→验证）。
- **扫描舵机并实时显示状态**：按范围扫描在线舵机，并周期读取 `0x38` 起始的 8 字节（位置/速度/负载/电压/温度）。
- **重置中置位 / 状态重置**：
  - 中置位：向 `0x2A` 写入目标位置=2048（并带速度/时间）。
  - 状态重置：发送 `RESET(0x0A)` 指令。

## 使用要求

- **桌面版** Chrome / Edge
- 站点必须是 **HTTPS**（或 `http://localhost`）
- 舵机必须**外接电源**（例如 6V+），USB 串口不供电
- 串口线序：**TX ↔ RX 交叉**，且 **GND 必须共地**
- 流水线/改号模式建议（且默认假设）任意时刻总线上只有 **1 个舵机**（广播写入会影响所有连接的舵机）

## 快速部署（GitHub Pages）

1. 新建一个仓库（Public/Private 均可），例如 `servo-id-configurator`
2. 上传 `index.html` 到仓库根目录
3. 仓库 Settings → Pages：
   - Source: Deploy from a branch
   - Branch: `main` / `(root)`
4. 打开 GitHub 提供的 HTTPS 链接即可使用

## 本地开发与测试

Web Serial 需要安全上下文，本地可用：

```bash
python -m http.server 8000
```

然后用 Chrome 打开：

- http://localhost:8000

## 免责声明

本工具会向广播地址（0xFE）写入参数。若同时连接多个舵机，可能导致多个舵机被同时改写。
请确保一次只接一台舵机，并确保供电与线序正确。
